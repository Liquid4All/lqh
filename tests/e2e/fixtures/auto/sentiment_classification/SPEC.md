# Sentiment Classification (3-class)

## Task
Classify the sentiment of a short text as one of: `positive`, `negative`,
or `neutral`. Output the label and nothing else.

## Input
- One to three sentences of English text from product reviews, social
  media posts, or short messages.

## Output format
A single word — exactly `positive`, `negative`, or `neutral`. No
punctuation, no commentary, no quotes.

## Quality criteria
- Correct label per the obvious sentiment of the text.
- `neutral` for factual statements or balanced opinions.
- Robust to typos and informal language.

## Base model
Use the smallest available LFM.
